Docker Docker es un subsistema de virtualización que comparte con el anfitrión el kernet y el sistema de archivor. El anfitrión siempre es un sistema Linux. Si estamos en Windows o MAC, necesitare una máquina virtual windows virtualizada con VirtualBox o similar

Un software de máquinas virtuales necesita instalar el sistema operativo completo. Ejemplos de sistemas de Virtualización: Para saber más sobre el tema:

  • VirtualBox (para multiples plataformas)
  • Hyper-V (Para Windows)
  • Parallels (Para MAC)
  • VMWare (para múltiples plataformas)
  • QEMU (Para Linux)
  • (Para UNIX/Linux)

      • LXD
      • Docker

Dockefile

Um dockerfile es un recta para crear una imagen para una aplicación particular de Docker. De este modo podemos tomar una imagen base, instalarle programas, configurar esos programas y dejarla preparada para crear contenedores a partir de ella.
